# From Dirt to GPUs

Field notes from inside the data center buildout.

This repository contains the site code for From Dirt to GPUs. The live project
is published at:

https://www.dirttogpus.com

## Development

Run locally:

```powershell
npm start
```

Then open:

```text
http://127.0.0.1:8000/
```

Environment variables are managed outside the repository. Do not commit local
`.env` files, subscriber data, backups, passwords, API keys, or production
database URLs.
